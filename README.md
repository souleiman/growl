GrOwl
=========
A web crawler written in Go. Get it? GrOwl :D

Requirements
------------
* Go 1.3.1+
* docopt
* html package from code.google.com

## Installation
        # Use go get to install growl
        go get github.com/souleiman/growl

Usage
------------
        > growl -u <url>
